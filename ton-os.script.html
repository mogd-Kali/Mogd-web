TON OS - Загрузка

Описание системы

TON OS - это не полноценная операционная система, а всего лишь bash-скрипт, который эмулирует некоторые функции операционной системы в терминале Linux.

Основные особенности:

  • Простой интерфейс командной строки
  • Основные команды для работы с файлами и системой
  • Минимальные требования к ресурсам

Установка

Для установки TON OS выполните следующие шаги:

  1. Убедитесь, что установлен wget:
  2. Kali Linux / Ubuntu / Debian:

    sudo apt install wget

  3. Termux:

    pkg install wget

  4. Arch Linux:

    pacman -S wget

  5. Скачайте скрипт установки:

    wget https://raw.githubusercontent.com/mogd-Kali/TON-OS/refs/heads/main/ton%20os.sh

  6. Запустите скрипт:

    bash ton os.sh
